Python Jupyter Notebook Extended Tutorial

Introduction

Jupyter Notebook is a powerful tool for interactive computing. It allows users to write and execute Python code in a web-based interface, making it ideal for learning, teaching, and conducting exploratory data analysis. In this extended tutorial, we will cover the essentials of Jupyter Notebook, focusing on how to effectively write, execute, and manage Python code, as well as explore its advanced features.

1. Launching Jupyter Notebook

To start Jupyter Notebook, open your terminal or command prompt and type:

jupyter notebook

This command will launch the Jupyter Notebook server and open the interface in your default web browser. If it doesnâ€™t open automatically, you can manually navigate to http://localhost:8888.

2. Creating and Organizing Notebooks

From the Jupyter Notebook dashboard, you can create a new notebook by clicking on New and selecting Python 3. Use meaningful names for your notebooks to keep them organized. You can rename a notebook by clicking on its title at the top of the page.

3. Working with Cells

3.1 Types of Cells

  • Code Cells: For writing and executing Python code. These cells display the output below them after execution.
  • Markdown Cells: For adding formatted text using Markdown syntax. These are useful for adding explanations and documentation.

3.2 Running Cells

To execute a code cell, press Shift + Enter. To run all cells in a notebook, go to Cell in the menu bar and select Run All.

4. Writing Python Code in Jupyter Notebook

4.1 Basic Python Code

# Example: Simple calculation
x = 10
y = 20
print("Sum:", x + y)

4.2 Using Libraries

Import libraries as you would in a regular Python script. Example:

import numpy as np
import matplotlib.pyplot as plt

# Generate data
data = np.linspace(0, 10, 100)
plt.plot(data, np.sin(data))
plt.show()

5. Markdown Syntax for Documentation

Markdown cells allow you to add formatted text. Here are some common Markdown elements:

  • Headings: Use # for headings. 
    # Heading 1
## Heading 2
### Heading 3
  • Lists: Create bullet or numbered lists. 
    - Bullet item 1
- Bullet item 2

1. Numbered item 1
2. Numbered item 2
  • Inline Code: Use backticks for inline code (`example`).
  • Links: [text](URL)

6. Useful Jupyter Notebook Features

6.1 Magic Commands

Magic commands enhance functionality in Jupyter Notebooks. Examples:

%timeit 2 + 2   # Measure execution time of code
!pip install pandas  # Run shell commands

6.2 Widgets

Interactive widgets can make notebooks more dynamic. Example:

from ipywidgets import interact

def greet(name):
    return f"Hello, {name}!"

interact(greet, name="World");

6.3 Exporting Notebooks

Save your work in various formats by going to File → Download as. Options include .ipynb, .html, and .pdf.

7. Advanced Tips

7.1 Keyboard Shortcuts

Boost productivity with shortcuts. Common ones include:

  • Ctrl + Enter: Run the current cell.
  • Shift + Enter: Run the current cell and move to the next.
  • Esc + A: Insert a cell above.
  • Esc + B: Insert a cell below.

7.2 Version Control with Notebooks

Use Git to track changes in notebooks. Convert them to a more Git-friendly format with nbconvert.

8. Practice Exercises

Try these exercises to get hands-on experience:

  1. Create a notebook that calculates and plots the Fibonacci sequence.
  2. Use a Markdown cell to document your code with headings and bullet points.
  3. Generate a scatter plot using matplotlib and annotate points with labels.

Note: If you encounter any issues, refer to the official Jupyter Notebook documentation.
